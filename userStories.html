User Stories

1.1: Animations

Intro: As a player I want to see smooth animations in the game.

Elaboration: Animations will be a large part of the fighting aspect of the game. They must be visually appealing and semi-realistic.

Constraints: none

Effort Estimation: 20 person hours

Acceptance Test: Create sprites for a specific task and code specific task to showcase animation.

1.2: Tutorial

Intro: As a player, I want to be offered an tutorial page and a move-list so that I learn how to play.

Elaboration: Each game has individual controls and commands. A tutorial will be useful to help users navigate the game mechanics.

Constraints: none

Effort Estimation: 2 person hours

Acceptance Test: Create "tutorial" button next to "play" button (on the main menu) to offer more usability than an in-game walkthrough.

1.3: Balance

Intro: As a player, I want to see the fighting types balanced against each other so that the gameplay is fair.

Elaboration: The game will have different types of fighters. They should all be equally powerful in their own way so that the matches rely on users' strategies.

Constraints: none

Effort Estimation: 8 person hours

Acceptance Test: Let players demo the game and test that they are all generally balanced against each type.

1.4 Items

Intro: As a player, I expect items to work as advertised so that I can use them to my advantage.

Elaboration: Each item has a different purpose. If I decide to use one of the items, I expect it to do what it's supposed to.

Constraints: none

Effort Estimation: 20 person hours

Acceptance Test: Players will demo the fights and try each item to ensure it works.
